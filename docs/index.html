Offensive Security

Red Team

Атакующая безопасность — это практика тестирования собственных систем путём имитации действий злоумышленников.

  • Пентестинг (тестирование на проникновение)
  • Социальная инженерия
  • Анализ уязвимостей
  • Эксплуатация систем
  • Red Team операции
$ nmap -sV target.com
Defensive Security

Blue Team

Защитная безопасность — это практика защиты систем и данных от киберугроз.

  • Мониторинг угроз (SIEM)
  • Реагирование на инциденты
  • Настройка файрволов
  • Шифрование данных
  • Управление доступом
$ sudo ufw enable
